Primary AWS Services

When designing on AWS, there are core services that form the backbone of most solutions. These services enable developers, architects, and businesses to build scalable, secure, and cost-effective cloud applications.

Key AWS Services Overview

  • EC2 (Elastic Compute Cloud)
  • S3 (Simple Storage Service)
  • EBS (Elastic Block Store)
  • EFS (Elastic File System)
  • RDS (Relational Database Service)
  • VPC (Virtual Private Cloud)
  • ELB (Elastic Load Balancing)
  • Route 53 (DNS and Health Checking)
  • IAM (Identity and Access Management)
  • CloudWatch (Monitoring and Logging)

Detailed Service Descriptions

AWS Identity and Access Management (IAM)

IAM helps securely control access to AWS resources. Use IAM to define who is authenticated and authorized to use resources. Always avoid using the root account for daily tasks—create users with least privilege.

Amazon EC2

Provides scalable compute capacity in the cloud. You can launch as many or as few virtual servers as needed, configure networking and security, and manage storage. EC2 enables scaling up or down to meet demand cost-effectively.

Amazon ECS & EKS

ECS (Elastic Container Service) is a highly scalable, high-performance container orchestration service. EKS (Elastic Kubernetes Service) is a managed Kubernetes service simplifying deployment and management of containerized apps.

Amazon ECR

Elastic Container Registry is a fully managed Docker container registry, making it easy to store, manage, and deploy container images securely and at scale.

Amazon S3

Industry-leading object storage with unmatched durability, availability, and scalability. Use S3 for data lakes, backups, archives, and application data storage.

Amazon EBS

Provides block-level storage volumes for EC2. Persistent and reliable, EBS volumes can be attached to instances and behave like raw, unformatted block devices.

Amazon EFS

Provides simple, serverless, elastic file storage that grows and shrinks automatically as you add and remove files, eliminating the need for capacity management.

Amazon RDS

Makes it easy to set up, operate, and scale a relational database in the cloud. Supports many database engines, provides automated backups, patching, and scaling.

Amazon VPC

Allows launching AWS resources into a virtual network defined by you. Resembles traditional networking but benefits from AWS's scale and flexibility.

Elastic Load Balancing (ELB)

Automatically distributes incoming traffic across multiple targets in one or more Availability Zones. Ensures only healthy targets receive traffic.

Amazon Route 53

Highly available and scalable DNS web service. Supports domain registration, DNS routing, and health checking to ensure reliable user access.

Amazon CloudWatch

Monitors AWS resources and applications in real time. Collects metrics, sets alarms, and enables automated responses to operational changes. Custom dashboards offer system-wide visibility.
